# Cadastro Inteligente

Projeto de um formulário de cadastro com validação em tempo real desenvolvido com HTML, CSS e JavaScript.

## 📌 Funcionalidades

- Validação de campos obrigatórios (nome, e-mail, senha e confirmação)
- Validação em tempo real utilizando o evento `blur`
- Mensagens de erro específicas para cada campo
- Feedback visual com estados de erro e sucesso
- Indicador de força da senha (fraca, média, boa e forte)
- Confirmação de senha
- Simulação de envio com loading
- Layout responsivo (mobile-first)

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git e GitHub

## 🎯 Objetivo

Praticar a criação de formulários modernos com validação em tempo real, aplicação de boas práticas de interface e versionamento de código utilizando Git.

## 🚀 Como executar o projeto

1. Baixe ou clone o repositório
2. Abra a pasta do projeto
3. Abra o arquivo `index.html` no navegador
